# TryHackMe SOC Analyst Writeups

A working portfolio of TryHackMe room writeups documenting hands-on practice across SIEM operations, incident response, network forensics, and threat hunting. Each writeup is structured to map lab activities to real SOC analyst responsibilities.

## About Me

I am working towards a SOC Analyst role in Australia, currently studying for CompTIA CySA+ (Security+ certified, February 2026). This repository is where I document what I learn so it stays useful past the point of completing a room.

## Repository Structure

Each room sits in its own folder with a focused README covering:

- Room overview and scenario
- Skills and tools demonstrated
- Key concepts and walkthrough of meaningful tasks
- How the lab work connects to a SOC analyst's day-to-day
- Personal takeaways
